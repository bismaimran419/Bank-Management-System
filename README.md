# Bank Management System (C++)

A lightweight **Bank Management System** built using core **C++** programming and **File Handling**. This project focuses on structured programming to handle user account operations and ensure data persistence without complex databases.

## Features
* **Account Creation:** Register new users with account holder names, account numbers, and secure pins.
* **Persistent Storage (File Handling):** Saves all account information dynamically to text files (`acount.txt`), so the data remains safe even after exiting the program.
* **Banking Operations:** 
  * Check and view account details.
  * Deposit money to update balances.
  * Withdraw money with automatic validation for insufficient balance.

##  Tech Stack & Concepts Used
* **Language:** C++
* **Data Persistence:** File Handling (`ofstream` / `ifstream` streams)
* **Core Concepts:** User-defined structures (`struct`), Functions, Passing by Reference (`&`), and File I/O operations.
